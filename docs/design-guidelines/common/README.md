# Common Design Guidelines

Документы, общие для backend и frontend (не дублируются в `go/` и `vue/`).

- `docs/design-guidelines/common/1_check_list.md` — обязательный чек-лист перед PR.
- `docs/design-guidelines/common/2_project_architecture.md` — зоны, границы ответственности, структура репо.
- `docs/design-guidelines/common/3_design_principles.md` — DDD/SOLID/DRY/KISS/Clean Architecture.
- `docs/design-guidelines/common/4_libraries_reusable_code_requirements.md` — правила выноса кода в `libs/*` (в т.ч. `libs/vue|ts|js`).
