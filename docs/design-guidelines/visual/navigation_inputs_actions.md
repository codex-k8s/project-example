# Навигация, ввод и действия

## 1) Навигация

Левый сайдбар:
- Содержит основные разделы, сгруппированные по смыслу.
- Активный пункт выделяется мягкой подложкой, акцентом и/или маркером.
- Иконки — простые, линейные, одинаковой толщины.

Верхняя зона (если есть):
- Для глобальных действий: выбор проекта/пространства, дата‑диапазон, экспорт, настройки.
- Не перегружать: максимум полезного, минимум «шума».

## 2) Навигация на узких устройствах

- Сайдбар всегда в виде drawer: скрыт по умолчанию, открывается по иконке (hamburger) и закрывается
  крестиком, тапом по затемнению, свайпом назад/влево (если уместно).
- Drawer занимает большую часть ширины, но оставляет тонкую «полосу контекста».
- Верхняя панель (app bar) — единственная зона глобальных контролов: название/контекст + кнопка меню/закрытия.

## 3) Контекст и переключатели на mobile

- Переключатели «проект / период / группировка / экспорт» группируй в одну строку сверху.
- Второстепенное убирай в overflow‑меню (⋯) или bottom sheet.
- Табы (Dashboard / Docs / API) на mobile делай как сегментированную панель
  или горизонтальный скролл‑ряд, если вкладок больше 3.

## 4) Ввод и «командная строка» (prompt‑bar)

Паттерн: одна главная строка ввода как центр сценария.

- Инпут крупный, заметный, с мягкой рамкой/подложкой.
- Справа/внутри — иконки вторичных действий (микрофон, отправка).
- Слева — контекст (файлы/репо/ветка/режим) через компактные селекты.
- Дополнительные опции показывай через выпадающее меню или «чипы‑подсказки» рядом.

На узких экранах:
- Используй нижнюю панель ввода, закреплённую снизу.
- Корректно учитывай safe area и поднимай панель над клавиатурой без скачков.
- Если есть режимы (например, «Думаю»), показывай их чипом внутри зоны ввода с быстрым снятием (×).

## 5) Кнопки и действия

Иерархия действий:
- Primary — одно главное действие на экране (например, «Создать», «Run», «Send»).
- Secondary — рядом, но визуально спокойнее.
- Tertiary/Ghost — для редких/вспомогательных.

Формулировки:
- Действия — глаголами: «Создать», «Экспорт», «Скопировать».
- Избегай эмоциональных формулировок и восклицаний.
