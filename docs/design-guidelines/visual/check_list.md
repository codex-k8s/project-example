# Чек‑лист качества для агента

Перед тем как считать дизайн‑решение готовым, проверь:

- [ ] На экране есть одно главное действие (primary).
- [ ] Иерархия заголовков и отступов делает страницу сканируемой.
- [ ] Цвет используется экономно: акцент + семантика, без «радуги».
- [ ] Таблицы: фильтры/поиск сверху, длинный текст обрезается и раскрывается.
- [ ] Пустые состояния объясняют, что делать дальше.
- [ ] Все кликабельные элементы имеют понятные названия и состояния (hover/active/disabled).
- [ ] Вёрстка не «разъезжается» на узких/широких экранах.
- [ ] На узких экранах сайдбар превращается в drawer, а глобальные контроли — в app bar.
- [ ] Основное действие доступно одной рукой, композер учитывает safe area и клавиатуру.
- [ ] Таблицы на mobile превращены в карточки‑строки, детали доступны по раскрытию.
- [ ] Фильтры/настройки на mobile доступны через bottom sheet или отдельный экран.
