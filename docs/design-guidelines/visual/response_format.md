# Рекомендуемый формат ответа агента (UI)

Если агент проектирует или предлагает интерфейс, отвечать структурно:

```markdown
## Страница: <Название>

### Цель
Коротко: что пользователь делает на этой странице.

### Компоновка
- Левый сайдбар: <разделы/группы>
- Верхняя зона: <проект/дата/экспорт…>
- Основной контент: <таблица/карточки/герой-блок>

### Компоненты
- <список ключевых компонентов>
- Состояния: empty/loading/error/success

### Поведение
- Фильтры/поиск
- Раскрытие деталей
- Копирование/экспорт

### Визуальные правила
- Отступы по шкале 4/8
- Один акцентный цвет + семантика
- Тонкие разделители, мягкие карточки
```
